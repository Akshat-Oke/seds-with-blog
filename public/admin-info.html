SEDS - Admin Info

This page is meant for people at SEDS who want to publish blog articles on this website.

Note: The urls are represented in the form /path. This means that you have to append the url (path) to the base url (host) of the website. For example, if you want to access the /admin page, you have to go to https://seds-bphc.vercel.com/admin.

Getting started /admin

Obtain the email and password for the admin account from your division head. Go to the admin page at /admin and log in.

The UI is terrible, I know
You should see a sign out button in the navbar on logging in

Creating an article /admin/create

Go to /admin/create and enter the title of the blog. Currently titles cannot be changed from the website frontend. The title will be used as the URL slug for the blog, so make sure it's unique.

Example: If the title is Intro to Archangel, the url will be /posts/intro-to-archangel. There is presently no checking done to assert that the title is unique.

On entering the title, you will be redirected to the post edit page.

Post edit page /posts/{slug}

Ensure that you are logged in as the admin, otherwise you will see a link to go back instead.

To edit a particular post, click the pen icon in the bottom right of the page while viewing the article (/posts/article-slug page).

Post content

Write your post in the Markdown format. You can use any Markdown editors (like this) or type it manually. Paste the markdown in the textbox under the title Content. Enter the author name above as well.

Cover image

The cover image is what will be shown on the blog home page at /posts as article thumbnails. To upload an image, click on the Upload Image button and select your image. You will then see the url of the image stored in the database, in this format: ![alt](download-url-here-copy-this). Copy the url from the download-url-here-copy-this part of the format and paste it in the Cover Image field.

Inserting images

To insert inline images in the blog article, upload the image as mentioned above and copy paste the whole returned format. Change the alt string to the image's description (alt text on hover).

Publishing

To publish, scroll the post edit page to the bottom and check the checkbox that says Published and click Save Changes.

If you uncheck the Published checkbox, the post will not show up in the blog thumbnail grid but can be viewed by manually going to its url.

Preview

You can preview your article on the same edit page by clicking the Preview button under "Tools". To view the live version as it would appear for users, click the Live view button.

Note: The style in the preview is not the same as the live version. The preview is meant to give you an idea of how the article will look like.
